# 🤫 PUPEL - Stealth P2P Chat

**PUPEL** is a privacy-focused, peer-to-peer (P2P) messaging tool designed by **Marascan**. It operates with **zero servers** and **zero logs**, ensuring your conversations stay between you and your peer.



---

### 🚀 Key Features
- **No Infrastructure:** No central server, no databases, no sign-ups.
- **True Privacy:** Messages are sent directly between browsers using WebRTC.
- **Zero Footprints:** Once you close the tab, the session and messages vanish forever.
- **Cross-Platform:** Works perfectly on **Mobile (PWA)** and **Desktop Browsers**.
- **Soft Matrix UI:** Eye-care dark theme designed for long-stealth sessions.

---

### 📱 How to Install (Mobile & Desktop)

PUPEL is a **Progressive Web App (PWA)**. You don't need an App Store to install it.

#### On Mobile (iOS/Android):
1. Open the link in your browser.
2. Tap the **Share** or **Menu** button.
3. Select **"Add to Home Screen"**.
4. Launch **PUPEL** directly from your apps.

#### On Desktop (Chrome/Edge):
1. Visit the link.
2. Click the **Install Icon** (monitor with arrow) in the URL bar.
3. Click **Install**.

---

### 🛠️ Tech Stack
- **WebRTC** (via PeerJS) for direct P2P connection.
- **Service Workers** for offline PWA support.
- **Vanilla JS** (Obfuscated for logic protection).
